# 박은미 포트폴리오

금융 IT 감사·내부통제 전문가 박은미의 풀스택 포트폴리오 프로젝트입니다.

## 📱 프로젝트 구성

### 1. 홈페이지 (웹)
- **파일**: `homepage.html`
- **설명**: 풀 포트폴리오 페이지
- **기능**: 프로필, 전문분야, 대표경험, 경력타임라인, 상훈·자격·교육, 연락처
- **특징**: 반응형 디자인, 미니멀 우아한 스타일, 스티키 네비게이션

### 2. 모바일 앱 (웹 앱)
- **파일**: `app.html`
- **설명**: iOS/Android 최적화 모바일 앱
- **기능**: 프로필, 경력, 연락처, 전자명함 (QR코드)
- **특징**: 다크모드, 전자명함 저장/공유, 오프라인 지원

## 🚀 빠른 시작

### 로컬 실행
```bash
# 웹 서버 실행
python -m http.server 8000

# http://localhost:8000/homepage.html (홈페이지)
# http://localhost:8000/app.html (모바일 앱)
```
